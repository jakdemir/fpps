# Functional Programming Principles in Scala

Coursera course assignments for Functional Programming Principles in Scala (EPFL).

## Overview

This repository contains my solutions for the Functional Programming Principles course, the foundation of the Scala specialization on Coursera by École Polytechnique Fédérale de Lausanne (EPFL).

## Topics Covered

- Functions and evaluation
- Higher-order functions
- Data and abstraction
- Types and pattern matching
- Lists and collections

## Assignments

- `recfun/` - Recursion exercises (Pascal's triangle, parentheses balancing)
- `funsets/` - Functional sets
- `objsets/` - Object-oriented sets (Twitter data)
- `patmat/` - Pattern matching (Huffman coding)
- `forcomp/` - For-comprehensions (anagrams)

## Technologies

- Scala
- SBT (Scala Build Tool)
- ScalaTest

## Running

```bash
cd <assignment>
sbt test
```
